# CcrAi

A self-learning agent for ChuChuRocket using reinforcement learning.

## Setup

```bash
# Install dependencies
uv sync

# Activate environment
source .venv/bin/activate

# Run tests
uv run pytest

# Format code
uv run black src/
uv run isort src/
```

## Project Structure

- `src/` - Main source code
- `tests/` - Test files
- `docs/` - Documentation
